    Message Analytics¶

    The MessageQuery system provides filtering, analytics, and export capabilities for conversation histories. It gives you structured access to the messages exchanged between users, assistants, and tools during agent execution.

    Accessing Messages¶

    Every Agent exposes a messages property that returns a fresh MessageQuery instance:

    from mamba_agents import Agent

agent = Agent("gpt-4o")
agent.run_sync("Hello!")
agent.run_sync("What tools do you have?")

# Access the query interface
query = agent.messages

    The messages property is stateless each access creates a new MessageQuery over the current message history. This means you always see the latest state without stale references.

    Filtering Messages¶

    Use filter() to search messages by role, tool name, or content. Multiple arguments combine with AND logic.

    By Role¶

    # Get all user messages
user_msgs = agent.messages.filter(role="user")

# Get all assistant messages
assistant_msgs = agent.messages.filter(role="assistant")

# Get all tool result messages
tool_msgs = agent.messages.filter(role="tool")

    By Tool Name¶

    # Find messages related to a specific tool
read_file_msgs = agent.messages.filter(tool_name="read_file")

    This checks both assistant messages with tool_calls and tool result messages.

    By Content¶

    # Case-insensitive text search
error_msgs = agent.messages.filter(content="error")

# Regex search
import_msgs = agent.messages.filter(content=r"import\s+\w+", regex=True)

    Combined Filters¶

    # Assistant messages mentioning "python"
python_assistant = agent.messages.filter(role="assistant", content="python")

# Tool results from a specific tool containing "success"
success_results = agent.messages.filter(
    role="tool",
    tool_name="run_bash",
    content="success",
)

    Slicing and Selecting¶

    Retrieve specific subsets of messages by position:

    # First message
first = agent.messages.first()

# Last 5 messages
recent = agent.messages.last(n=5)

# Messages 2 through 5 (0-indexed, exclusive end)
middle = agent.messages.slice(start=2, end=6)

# All messages (equivalent to agent.get_messages())
all_msgs = agent.messages.all()

    Conversation Analytics¶

    Message Statistics¶

    The stats() method computes token and message count statistics:

    stats = agent.messages.stats()

print(f"Total messages: {stats.total_messages}")
print(f"Total tokens: {stats.total_tokens}")
print(f"Avg tokens/message: {stats.avg_tokens_per_message:.1f}")

# Breakdown by role
for role, count in stats.messages_by_role.items():
    print(f"  {role}: {count} messages")

for role, tokens in stats.tokens_by_role.items():
    print(f"  {role}: {tokens} tokens")

    You can also print the stats directly for a formatted summary:

    print(stats)
# Message Statistics
#   Total messages: 12
#   Total tokens:   1543
#   Avg tokens/msg: 128.6
#   Messages by role:
#     assistant: 4
#     tool: 3
#     user: 5
#   Tokens by role:
#     assistant: 890
#     tool: 203
#     user: 450

    Tool Call Summary¶

    The tool_summary() method provides analytics on tool usage across the conversation:

    summaries = agent.messages.tool_summary()

for info in summaries:
    print(f"Tool: {info.tool_name}")
    print(f"  Called {info.call_count} time(s)")

    for i, call_id in enumerate(info.tool_call_ids):
        print(f"  [{call_id}]")
        if i < len(info.arguments):
            print(f"    args: {info.arguments[i]}")
        if i < len(info.results):
            print(f"    result: {info.results[i][:100]}...")

    Conversation Timeline¶

    The timeline() method parses messages into logical conversation turns:

    turns = agent.messages.timeline()

for turn in turns:
    print(f"Turn {turn.index}:")

    if turn.system_context:
        print(f"  System: {turn.system_context}")

    if turn.user_content:
        print(f"  User: {turn.user_content}")

    if turn.assistant_content:
        print(f"  Assistant: {turn.assistant_content[:80]}...")

    for interaction in turn.tool_interactions:
        print(f"  Tool: {interaction['tool_name']}")
        print(f"    args: {interaction['arguments']}")
        print(f"    result: {interaction['result'][:80]}...")

    Each Turn groups related messages:

    • User prompt that initiated the turn
    • Assistant response
    • Tool call/result pairs from tool use within the turn
    • System context (attached to the first turn)

    Exporting Conversations¶

    Export messages in multiple formats using export():

    JSON¶

    json_str = agent.messages.export(format="json")

# With metadata (index and token counts)
json_with_meta = agent.messages.export(format="json", include_metadata=True)

    Markdown¶

    markdown = agent.messages.export(format="markdown")
print(markdown)
# ### User
#
# Hello, what tools do you have?
#
# ---
#
# ### Assistant
#
# I have several tools available...
#
# **Tool Call: list_tools**
#
# ```json
# {}
# ```
#
# **Result:**
#
# ```
# read_file, write_file, run_bash
# ```

    CSV¶

    csv_str = agent.messages.export(format="csv")

# With custom content truncation
csv_str = agent.messages.export(format="csv", max_content_length=200)

    The CSV format includes columns: index, role, content, tool_name, tool_call_id, token_count.

    Dict¶

    enriched = agent.messages.export(format="dict")

for msg in enriched:
    meta = msg["_metadata"]
    print(f"[{meta['index']}] {meta['role']}: {meta['token_count']} tokens")

    Exporting a Subset¶

    Pass a filtered list to export only specific messages:

    # Export only tool-related messages as markdown
tool_msgs = agent.messages.filter(role="tool")
tool_export = agent.messages.export(format="markdown", messages=tool_msgs)

    Working Without Token Counting¶

    MessageQuery works gracefully when no TokenCounter is available. All analytics methods return valid results token counts default to zero:

    from mamba_agents import Agent, AgentConfig

# Context tracking disabled
agent = Agent("gpt-4o", config=AgentConfig(track_context=False))

# messages property returns empty MessageQuery
query = agent.messages
stats = query.stats()  # Returns empty MessageStats

    Standalone Usage¶

    You can use MessageQuery independently of the Agent facade:

    from mamba_agents.agent.messages import MessageQuery

# Work with raw message dicts
messages = [
    {"role": "user", "content": "Hello"},
    {"role": "assistant", "content": "Hi there!"},
    {"role": "user", "content": "Run a command"},
    {
        "role": "assistant",
        "content": None,
        "tool_calls": [{
            "id": "call_1",
            "function": {"name": "run_bash", "arguments": '{"command": "ls"}'},
        }],
    },
    {"role": "tool", "tool_call_id": "call_1", "name": "run_bash", "content": "file1.py\nfile2.py"},
    {"role": "assistant", "content": "I found two files."},
]

query = MessageQuery(messages)

# All filtering, analytics, and export methods are available
user_msgs = query.filter(role="user")
stats = query.stats()
timeline = query.timeline()
export = query.export(format="markdown")

    To enable token counting in standalone mode, pass a TokenCounter:

    from mamba_agents.agent.messages import MessageQuery
from mamba_agents.tokens import TokenCounter

counter = TokenCounter()
query = MessageQuery(messages, token_counter=counter)

stats = query.stats()
print(f"Total tokens: {stats.total_tokens}")  # Actual token counts
